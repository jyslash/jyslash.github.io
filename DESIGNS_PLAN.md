# Designs Section — Project Planning

This file tracks cover image status for all projects in `_designs/`. Source of truth for project list is `Project Candidates.md`.

---

## Status Key

| Symbol | Meaning |
|--------|---------|
| ⬜ | Pending — no cover image yet |
| 🔍 | Under review — candidates copied, picking cover |
| ✅ | Done — cover chosen, MD file updated |
| ❌ | Dropped — decided not to include |

---

## Cover Image Guidelines

- **Format**: PNG or WebP preferred, JPG acceptable
- **Ratio**: 16:9 (1280×720 or 960×540 minimum)
- **Pick criteria**: strongest visual moment — actual UI work or composition shots, avoid generic mockups
- **Naming**: `[slug]/cover.[ext]` e.g. `line-design-system/cover.png`
- **Destination**: `assets/images/projects/[slug]/cover.[ext]`
- **When ready**: update `cover:` field in the corresponding `_designs/*.md`
